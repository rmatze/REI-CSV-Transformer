MLS APN Processor

San Bernardino, CA

How it works

  1. Download the template CSV or use an existing you've already created
  2. Copy in your list of APNs in the first column. Makes sure the column header is named APN. If you have a CSV that already contains a column with the header APN, you can use that.
  3. Upload the CSV file with your APNs
  4. The APN format should be correct for San Bernadino, but update as needed. The hashes (#) will be replaced with the APN, and the dashes (-) will print between the hashes.
  5. Click "Process CSV" and download the new CSV file that contains formatted APNs. This first version will be for San Bernadino, CA so those columns will be add automatically.
